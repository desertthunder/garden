# Bentham's Scale

- Intensity
- Duration
- Certainty
- Propinquity (how soon it will happen)
- Fecundity (how much pleasure)
- Purity ($\frac{Pleasure}{Pain}$)
- Extent (how many people benefit)
    - If you're acting publicly, spread as much pleasure you can
